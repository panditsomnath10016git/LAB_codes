###  Using "phc_codes" folder as a package in python
copy "phc_codes" folder to the python package site (in windows is usually at - 'C:\Users\somna\AppData\Local\Programs\Python\Python31x\Lib') to import from anywhere. Otherwise you can keep the folder in your working directory.

Now in any of your codes you can import the fuctions as import phc_codes.xxx as pc
